# mc-sheetmusic2noteblock ドキュメント

ピアノ楽譜をアップロードすると、Minecraft の音符ブロック(ノートブロック)+レッドストーン回路で
その曲を演奏する装置の「組み立て設計書」をブラウザに表示する Web アプリ。

| [RESEARCH.md](RESEARCH.md) | 調査結果(Minecraft 音符ブロック/レッドストーン仕様、量子化の数理、OMR 候補比較) |
| [DESIGN.md](DESIGN.md) | システム設計(アーキテクチャ、リポジトリ構成、API、データモデル、フロント画面) |
